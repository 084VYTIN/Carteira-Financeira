# Carteira Financeira
Projeto de controle financeiro pessoal desenvolvido para estudos de desenvolvimento Full Stack, utilizando Ruby com Sinatra no back-end e JavaScript no front-end.

### 🚀 Como rodar o projeto
Pré-requisitos: Certifique-se de ter o Ruby instalado.

## Instalar dependências:

Bash
gem install sinatra
3. **Iniciar o servidor:**
ruby app.rb
Acessar: Abra o navegador em http://localhost:4567

### 🛠 Tecnologias utilizadas
Back-end: Ruby, Sinatra

Front-end: HTML5, CSS3, JavaScript (Fetch API)
